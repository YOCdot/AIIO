## 1.后端

后端端口 `http://localhost:8000/`

```shell
conda activate backend

python manage.py makemigrations

python manage.py migrate

python manage.py runserver
```

## 2.前端

前端端口 `http://localhost:5173/`

```shell
cd AiioFE

npm install

npm run dev
```
